# BingoBeat
Bingo Musical.

## Aviso legal
- Esta app **no almacena ni distribuye archivos de audio** (MP3 u otros formatos).
- Esta app **no reproduce música dentro de la aplicación**.
- Los títulos de canciones se usan únicamente como referencia para la dinámica del juego bingo musical.
- La reproducción musical, si existe, ocurre fuera de la app y bajo responsabilidad del host/organizador.
